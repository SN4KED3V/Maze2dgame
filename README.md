Escape the depths of the crypt in this grid-based Dungeon Maze crawler. Built using Python's `turtle` graphics, this game combines classic maze navigation with RPG-style exploration, featuring hidden exits, collision-based physics, and a modular level-design system.

---

## 🚀 Quick Start

### 1. Prerequisites
You only need **Python 3.x** installed. The project relies on the native `turtle` library, so no external `pip` installations are necessary.

> **Note for Linux users:** If the dungeon window fails to initialize, ensure the Tkinter backend is installed:
> ```bash
> sudo apt-get install python3-tk
> ```

### 2. Enter the Dungeon
Clone the repository and launch the quest:
```bash
# Clone the repository
git clone [https://github.com/SN4KED3V/Dungeon-Maze-Python.git](https://github.com/SN4KED3V/Dungeon-Maze-Python.git)

# Enter the project directory
cd Dungeon-Maze-Python

# Start the crawl
python dungeon_maze.py
